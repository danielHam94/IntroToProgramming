Database Questions

  • Primary key is a unique value that represents a whole row. Every row will be differentiated by the primary key.
  • Foreign key is a type of key that exists in one table but refers to the primary key in another table. This is what allows us to link tables together.
  • Tuple is a row differentiate by primary key. Attributes is the (column) foreign key that refers to the primary key in table.
  • SQL stands for Structured Query Language. It is composed of short lines of codes that are grouped together. SQL is a database language and the standard language for Relational Database Management Systems.
  • A key example could that could be linked is a table of different sections of a certain course and a table of the different sections that contain the names of students registered for the course. Linking a primary key and foreign key will allow changes in one table to effect the whole database.
