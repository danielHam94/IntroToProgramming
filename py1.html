Name and define the programming terms we learned today.
Function-a type of procedure or routine that carries out a task
Argument-a value that is passed to a function
String-a sequence of characters
Variable-a container, given a name, that represents a value
Integer-a whole number (number without a decimal point)
Why is it important to keep track of data types?
Because there can be errors in using different data types. Spelling is really important.
